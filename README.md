# Evaluación 1 – Programación 3

## Objetivo

El objetivo de esta evaluación es consolidar los conocimientos adquiridos en las
primeras unidades del cursado (**HTML**, **CSS**, **JavaScript** y **TypeScript**), mediante la
evolución del proyecto Food Store hacia una aplicación frontend más dinámica e
interactiva.

---

## Descripcion del proyecto

Consiste de una aplicacion online de e-commerce sencillo con registro e ingreso de usuarios autenticados persistidos en el almacenamiento local del navegador. 
Esta entrega cuenta con la complecion del front-end de siguientes secciones:
- Sección principal (Home)

Catálogo de productos con filtro por categoría y barra de búsqueda por de producto por nombre.
- Sección Carrito (Cart)

Resumen de la compra con subtotales y total y detalle de los productos agregados, con la posibilidad de sumar o restar unidades, eliminar productos o vaciar el carrito completamente.

---

## 🚀 Requisitos, Instalación y Uso

Se recomienda usar `pnpm` como gestor de paquetes y contar con version de `node.js` 22 o superior.

### 1. Instalar pnpm

```bash
npm install -g pnpm
```

### 2. Instalar Dependencias del Proyecto

```bash
pnpm install
```

### 3. Ejecutar el Proyecto (servidor de desarrollo de VITE)

```bash
pnpm dev
```

La aplicación estará disponible en la URL que aparezca en la terminal (generalmente `http://localhost:5173`).
